## Key Functions
1. **Audio Upload**  
   Upload your own audio files (e.g., songs, recordings) for processing.

2. **Separation Process**  
   The tool extracts:
   - **Vocals** ‚Äì the vocal track containing singing or speaking.  
   - **Background/Instrumental** ‚Äì the rest of the audio excluding vocals.

3. **Output Download**  
   After processing, you can download the separated tracks individually.
